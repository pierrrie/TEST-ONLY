# My App

React приложение с TypeScript, SCSS и Swiper

## Технологии

- React 19
- TypeScript
- SCSS
- Swiper
- GSAP
- Webpack

## Установка и запуск

1. **Клонируйте репозиторий**

   ```bash
   git clone <https://github.com/pierrrie/TEST-ONLY.git>
   cd my-app

   ```

2. **Установите зависимости**
   npm install

3. **Запуск в режиме разработки**
   npm run dev

4. **Сборка для production**
  npm run build
